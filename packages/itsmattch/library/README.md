## Instalacja

Główne repozytorium: https://github.com/mch300c/laravel

```bash
# Sklonuj główne repozytorium
git clone git@github.com:mch300c/laravel.git

# Przejdź do pobranego folderu
cd laravel

# Zainstaluj zależności
composer i

# Zainstaluj sail wybierając co najmniej bazę danych (np. mysql)
sail install

# Uruchom sail
sail up

# Wykonaj migracje
sail artisan migrate
```
